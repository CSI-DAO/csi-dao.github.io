Projekt

Wir schaffen eine faire Absicherung gegen Alltagsrisiken.

Das sind die Fokuspunkte unseres Forschungsvorhabens.

Autonomie

Im Gegensatz zu einer klassischen Versicherung soll der gesamte Prozess über eine Decentralized Autonomous Organization (DAO) abgewickelt werden. Es gibt also keine Mitarbeiter.

Gemeinschaft

Die regionalen Möglichkeiten der Genossenschafts-ähnlichen Strukturen einer DAO sollen eine direkte Beteiligung der Mitglieder an Entscheidungsprozessen ermöglichen.

Transparenz

Ziel ist die Umsetzung auf einer öffentlichen Blockchain. Jeder kann so nachvollziehen, wie Entscheidungen getroffen werden und es geschieht nichts hinter verschlossenen Türen.

Maschinelles Lernen

Zur vollautomatischen Risikoermittelung soll eine Data Mining Komponente entwickelt werden, sodass auch individuelle Prämien möglich sind.

Inklusion

Ziel ist eine faire & nachhaltige Möglichkeit zur Risikoabsicherung, bei der sich jeder beteiligen kann. Faktoren, wie Herkunft oder Name einer Person spielen dabei keine Rolle.

Nachhaltigkeit

Mögliche Gewinne der DAO sollen in gemeinnützige Projekte investiert werden. DAO-Mitglieder können entsprechende Vorschläge machen und für ihre Favoriten abstimmen.
